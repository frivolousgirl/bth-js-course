Module: js/kmom03.js

Interactive Button and Element Creation with Flip-Back This script creates an interactive button that, when clicked, dynamically generates and appends div elements to the document. Each generated element is styled with a background color and contains additional information such as a name, an image, and a year. The script uses data imported from "./modules/kmom03/info.js" to set the colors, persons' names, and years for each generated element. The button is initially centered on the screen, and its background color and text content change with each click. Each generated element has a class "box" and responds to click events by displaying additional information and an image. The elements also gain a border when hovered over. The script ends the button display when it cycles through all the colors. Additionally, the script allows flipping back the generated elements when clicking on them again, toggling between the initial state and the flipped state.
